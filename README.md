# Moses Welding Tools

Moses Welding Tools is a Python-based application for welders and small businesses.  
It includes:

- Basic Welding Calculator
- Project Cost Calculator
- Daily Welding Log with total earnings
- Advanced Moses Welding Calculator V2

**How to run:**  
1. Download `moses_welding_tools.py`  
2. Ensure `welding_log.txt` is in the same folder  
3. Run the program in Python (Pydroid, PC, or any Python IDE)  

**Features:**  
- Input validation for numbers  
- Persistent storage of daily jobs  
- Clear and professional menu
